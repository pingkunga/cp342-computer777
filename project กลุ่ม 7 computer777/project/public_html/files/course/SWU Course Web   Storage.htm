Storage

 

 4. 10 คะแนน  Access Time  Harddisk

 

 จากตัวอย่าง specification ของฮาร์ดดิสก์รุ่นหนึ่ง เป็นดังนี้  

750 GB HARD DRIVE [Serial-ATA-II, 3Gb, 7200 RPM, 16M Cache]

 

Description:
750 GB HARD DRIVE [Serial-ATA-II, 3Gb, 7200 RPM, 16M Cache]

Specification:
Interface: SATA 3.0Gb/s
Cache: 16MB
RPM: 7200 RPM
Average Seek Time: 11ms
Average Latency: 4.16ms

Data Transfer 100 Mbytes/sec

อธิบายความหมายของข้อกำหนดต่อไปนี้

ก.     RPM: 7200 RPM

 

ข.     Average Seek Time: 11ms

 

ค.     Average Latency: 4.16ms

 

ง.      Data Transfer 100 Mbytes/sec

 

จ.     แสดงวิธีการคำนวณ Access Time ของฮาร์ดดิสก์นี้
